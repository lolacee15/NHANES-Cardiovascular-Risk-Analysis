# NHANES Cardiovascular Risk Analysis

## Overview
This project analyzes cardiovascular risk factors using data from the National Health and Nutrition Examination Survey (NHANES). The goal is to explore patterns in health indicators and identify relationships that may contribute to cardiovascular disease risk.

## What I Did
- Cleaned and preprocessed large-scale health data using PySpark
- Explored key cardiovascular risk indicators such as age, BMI, and blood pressure
- Performed data analysis to identify trends and relationships in patient health outcomes
- Applied distributed data processing techniques to handle large datasets efficiently

## Tools Used
- Python
- PySpark
- pandas
- matplotlib

## Key Takeaways
This project strengthened my ability to work with real-world healthcare data, use distributed computing tools, and translate data into meaningful insights related to patient health outcomes.
